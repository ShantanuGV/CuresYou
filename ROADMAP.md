# Roadmap

This roadmap outlines the planned evolution of CurseYou.

---

## Version 1.0 — Core Engine

### Completed

* [x] Metadata indexing
* [x] Candidate retrieval
* [x] Beam Search optimization
* [x] Layout computation
* [x] Procedural stitching
* [x] Bézier bridge generation
* [x] Alpha compositing
* [x] PNG export
* [x] Modular backend

---

## Version 1.1 — Performance

Planned improvements:

* [ ] Multi-threaded optimization
* [ ] Faster metadata loading
* [ ] Memory usage optimization
* [ ] Improved candidate caching
* [ ] Rendering speed improvements

## Version 1.2 — Rendering

Planned improvements:

* [ ] Improved bridge realism
* [ ] Better blending masks
* [ ] Adaptive river width normalization
* [ ] Higher-quality compositing
* [ ] Optional SVG export

---

## Version 2.0 — Advanced Features

Future goals:

* [ ] Interactive editor
* [ ] Automatic dataset expansion
* [ ] GPU acceleration
* [ ] Real-time rendering preview
* [ ] Plugin architecture
* [ ] Web API
* [ ] Batch rendering
* [ ] Advanced optimization strategies

---

## Long-Term Vision

The long-term objective is to evolve CurseYou into a flexible procedural typography platform capable of composing high-quality river-based text while remaining modular, extensible, and well documented.
